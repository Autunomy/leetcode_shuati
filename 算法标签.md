# 语法

### 数组的下标和数据一起排序

```java
//nums是待排序数组 idx是需要排序的下标数组
Integer[] idx = new Integer[n];
for (int i = 0; i < n; ++i) {
    idx[i] = i;
}
Arrays.sort(idx, (i, j) -> nums[i] - nums[j]);
```
